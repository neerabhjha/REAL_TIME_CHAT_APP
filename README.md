## Features

- Register and Login User
- Authentication using JWT
- Search for chats and users
- Emoji Sharing
- Image Sharing 
- User Online Status 
- Message time 
- Update Profile with user image
- Pagination

## Tech Stack

**Client:** React, Redux, Tailwind

**Server:** Node, Express, Socket.io
